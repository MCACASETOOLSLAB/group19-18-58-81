BUSY WAITING

Definition :

                     Continuously testing a variable until some value appears is called busy waiting.

Conditional Synchronization

                     # A lock that uses busy waiting is called a spin lock .
                     # It should usually be avoided, since it wastes CPU time.
                     # Only when there is a reasonable expectation that the wait will be short is busy waiting used.


Issues with this critical section solution:

             # Busy wait (aka. polling): In a loop, continuously checking the value of variables

             # These processes are doing a busy wait until allowed into critical section

             # Busy waiting is not the same as a process being blocked.

             # Threads actively take CPU cycles when waiting for other threads to exit from critical section
